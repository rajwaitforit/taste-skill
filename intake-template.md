# Taste Intake Template

Use this template verbatim when onboarding a new user or project.

1) What are we curating for?  
2) What feeling should it give? (3-5 adjectives)  
3) Share 3 things you love + one-line why for each  
4) Share 3 things you dislike + one-line why for each  
5) Where should we sit on these spectrums?  
   - safe ↔ experimental  
   - mainstream ↔ niche  
   - polished ↔ raw  
   - luxury ↔ accessible  
6) Any hard constraints? (budget, region, exclusions)  
7) What output format do you want?  
8) What defines a “great hit” for you?
